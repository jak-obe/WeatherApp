# WeatherApp

## General info
This project is weather forecast app.

 ## Libraries used: 
* Jetpack Compose
* Room
* Retrofit2

## Setup
To run this project, install it on your android device.


## Features:
* Weather for today, with a short description.
* Forecast for next 7 days.
* Forecast for every hour today.
